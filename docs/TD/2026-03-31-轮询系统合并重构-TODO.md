    - `COMPACT_POLL_INITIAL_DELAY_MS` → `POLL_INITIAL_DELAY_MS`（150ms）
    - `compactT()` → `pollT()`（委托 `translateCompactText`）
    - `_handlePollError()` → 保留（3 次错误自动停止）
    - `_notifyNewEmailAndStop()` → 保留（验证码提取 + 剪贴板复制）
    - `stopCompactAutoPoll()` → `stopPoll(email, toastMsg, toastType)`
    - `stopAllCompactAutoPolls()` → `stopAllPolls()`
    - `startGlobalCountdown()` → 保留（1 秒倒计时，通过 `onPollTick` 回调更新 UI）
    - `pollSingleEmail()` → 保留（inbox + sentitems 并行 fetch）

    - 新增 `reapplyAllPollUI()` — 重新应用所有运行中轮询的 UI
    - 新增 `registerPollUICallbacks(callbacks)` — UI 回调注册接口

    - `updateCompactPollUI()` — compact-specific UI 更新

- [x] **P1-2**：`mailbox_compact.js` 改为调用 `poll-engine.js` API
  - 注册包含 `mailboxViewMode` 分发逻辑的 UI 回调：
    ```js

      onPollStart: function(email, maxCount) {
        if (mailboxViewMode === 'compact') updateCompactPollUI(email, 'polling', maxCount);
        else showStandardPollDot(email);
      },
      onPollStop: function(email) {
        if (mailboxViewMode === 'compact') updateCompactPollUI(email, 'stopped', null);
        else hideStandardPollDot(email);
      },
      onPollTick: function(email, remaining) {
        if (mailboxViewMode === 'compact') updateCompactPollUI(email, 'polling', remaining);
      },
      onNewEmail: function(email, summary) {
        if (mailboxViewMode === 'compact') updateSingleRowFromCache(email, summary);
      },
      onAccountCheck: function(email) {
        // 检查账号是否存在于缓存/DOM
      }
    });
    ```

## 注意事项

1. **Phase 1 是纯内部重构**，不应改变任何外部行为。如果 Phase 1 后测试不全绿，说明迁移有问题，应立即修复。
2. **Phase 2 有行为变更**（标准模式触发方式改变），需要人工验证。
3. **Phase 3 是设置层面的破坏性变更**，compact 字段被废弃。但由于 Phase 2 已经让引擎不再使用这些字段，影响可控。
4. 每个 Phase 完成后都应提交一个独立 commit，方便回滚。

### 关键代码路径

```
用户复制邮箱
  → emails.js: copyEmail() 派发 CustomEvent('email-copied')
  → mailbox_compact.js: window.addEventListener('email-copied', ...)
  → poll-engine.js: startPoll(email)
    → pollSingleEmail(email, state) [每 N 秒]
      → fetch /api/accounts/{email}/emails?folder=inbox
      → fetch /api/accounts/{email}/emails?folder=sentitems
      → 比较 baselineIds，发现新邮件 → _notifyNewEmailAndStop()
        → 验证码提取 + copyToClipboard + Toast
  → UI 回调分发 (mailbox_compact.js registerPollUICallbacks)
    → compact: updateCompactPollUI()
    → standard: showStandardPollDot() / hideStandardPollDot()
```
